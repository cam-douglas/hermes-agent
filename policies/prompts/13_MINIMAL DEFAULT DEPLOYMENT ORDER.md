SUB-PROMPT — MINIMAL DEFAULT DEPLOYMENT ORDER

When implementing this agentic company architecture, do not create the whole company at once.

Use this order:

1. Chief Orchestrator
2. Org Mapper / HR Controller
3. Product Director
4. Engineering Director
5. Operations Director
6. IT / Security Director
7. one Project Lead per active project
8. Workers on demand
9. Supervisors only when project complexity justifies them
10. Design, Commercial, Finance, and Admin only when actual workload appears

Always choose the smallest viable structure that can credibly deliver outcomes.
